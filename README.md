# sistema-de-atividades-eemti-jose-bezerra-de-menezes
<h1 align="center"> Sistema de Atividades</h1>
<p align="center">Caderno Digital Integrado para organização de matérias escolares da minha escola</p>

<hr>

<h2>Sobre o Projeto</h2>
<p>
Este projeto é um sistema web simples que funciona como um <b>caderno digital interativo</b>,
permitindo organizar e acessar atividades por disciplina de forma prática. :)
(tive que refazer)
</p>

<h2> Funcionalidades</h2>
<ul>
  <li>📂 Menu lateral com matérias</li>
  <li>🔍 Pesquisa de disciplinas</li>
  <li>🏠 Tela inicial (dashboard)</li>
  <li>📖 Organização por matérias</li>
  <li>📱 Interface responsiva</li>
</ul>
